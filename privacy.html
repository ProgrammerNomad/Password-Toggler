Privacy Policy

Last updated: November 6, 2025

TL;DR: Password Toggler does NOT collect, store, or transmit any user data. Period.

Data Collection

Password Toggler does not collect any user data. We do not track, store, or transmit:

  • Passwords or password field contents
  • Browsing history or URLs visited
  • Personal information of any kind
  • Usage statistics or analytics
  • Cookies or tracking identifiers

Local Storage

The extension uses Chrome's storage permission only to save your preferences locally on your device:

  • Selected toggle mode (Mouse Over, Double Click, Focus, or Ctrl Key)
  • Mouse over delay time setting
  • Language preference (auto-detected from browser)

These settings remain on your device and are never sent to any server or third party.

Permissions

Password Toggler requests minimal permissions:

  • storage - To save your toggle mode preferences locally
  • host_permissions - To access password fields on websites you visit

These permissions are used only for the extension's core functionality and nothing else.

Third-Party Services

Password Toggler does not use any third-party services, including:

  • No analytics (Google Analytics, etc.)
  • No crash reporting tools
  • No advertising networks
  • No remote servers or APIs

Offline Functionality

The extension works completely offline. All code runs locally in your browser with no internet connection required.

Open Source

Password Toggler is open source. You can review the complete source code at: https://github.com/ProgrammerNomad/Password-Toggler

Changes to This Policy

Any changes to this privacy policy will be reflected in the extension's GitHub repository and updated here.

Contact

For questions or concerns about privacy, please contact the developer via: GitHub Issues
